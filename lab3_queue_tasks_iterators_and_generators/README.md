# Лабораторная работа №3: Очередь задач: итераторы и генераторы

### Описание проекта
В 3-й лабораторной работе мы реализовали механизмы ленивой обработки данных. Теперь система способна обрабатывать огромные потоки задач, не загружая их целиком в оперативную память. Мы превратили простой сборщик задач в интеллектуальную очередь-конвейер.

**Ключевые изменения:**
* **Протокол итерации:** Класс `TaskQueue` реализует метод `__iter__`, что позволяет использовать его в циклах `for`, а также в функциях `list()`, `sum()`, `max()` напрямую.
* **Ленивые фильтры:** Фильтрация по статусу и приоритету реализована через генераторы. Задачи фильтруются только в момент обращения к ним («на лету»).
* **Эффективность памяти:** Система не хранит задачи в промежуточных списках. Используется делегирование итерации (`yield from`), что минимизирует потребление ресурсов при работе с большими JSONL-файлами.
* **Повторный обход:** Очередь спроектирована так, что её можно обходить многократно — каждый новый цикл итерации заново обращается к источникам данных.

### Что было изучено
* **Протоколы Iterable и Iterator:** Понимание разницы между объектом, по которому можно итерироваться, и самим итератором.
* **Генераторы и `yield`:** Создание функций, которые сохраняют свое состояние и возвращают значения порциями.
* **Конструкция `yield from`:** Эффективное делегирование итерации под-генераторам и источникам данных.
* **Lazy Evaluation (Ленивые вычисления):** Принцип выполнения операций только тогда, когда их результат действительно необходим программе.

### Установка и запуск

**Локальная установка зависимостей:**
```bash
pip install -r requirements.txt
```

**Запуск основной программы:**
```bash
python3 -m src.main
```

**Запуск тестов и проверка покрытия (не менее 80%):**
```bash
pytest --cov=src tests/
```

### Работа в Docker

**1. Сборка образа:**
```bash
docker build -t lab3-tasks .
```

**2. Запуск тестов и отчета о покрытии внутри контейнера:**
```bash
docker run --rm lab3-tasks
```

**3. Запуск демонстрации ленивой очереди:**
```bash
docker run --rm lab3-tasks python -m src.main
```
